# 🧟 Zombie Escape · English Survival

**Zombie Escape** es un videojuego educativo de supervivencia donde los jugadores eliminan zombies respondiendo preguntas de inglés (Top Notch Fundamentals, Units 7 y 8). Diseñado como una experiencia inmersiva estilo arcade, con pixel art moderno, efectos visuales y sonoros.

## 🎮 Características

- **150+ preguntas** de opción múltiple (Simple Present, Daily Activities, Household Chores, There is/are, How Many, etc.)
- **Sistema de vidas** (3 ❤️) y cronómetro (40s) con bonificaciones (+3s acierto, -5s error)
- **Ranking local** (Top 10) almacenado en `localStorage`
- **Interfaz de juego** con animaciones, partículas, niebla, lluvia y efectos de explosión
- **Totalmente responsive** (PC, tablet, móvil)
- **Sin servidor, sin APIs, sin frameworks** — HTML5 + CSS3 + JavaScript Vanilla

## 🚀 Cómo ejecutar

### Opción 1: Live Server (recomendado para desarrollo)
1. Clona o descarga este repositorio.
2. Abre la carpeta con **Visual Studio Code**.
3. Instala la extensión **Live Server**.
4. Haz clic derecho en `index.html` → **Open with Live Server**.

### Opción 2: GitHub Pages
1. Sube todos los archivos a un repositorio público en GitHub.
2. Ve a **Settings → Pages**.
3. Selecciona la rama `main` y la carpeta raíz (`/`).
4. Guarda y espera unos minutos. El juego estará disponible en `https://tuusuario.github.io/ZombieEscape/`.

### Opción 3: Doble clic en index.html
- Funciona, pero algunos navegadores pueden bloquear `localStorage` si se abre desde `file://`. Recomendamos Live Server o GitHub Pages.
